# musicLink

LangBot的点歌插件，可以选择音乐并下载高质量音乐。仅供开发者测试和学习使用，请勿用于商业或违规用途。所有数据来源于网络，如有版权问题请联系删除。

## 使用方法

通过LangBot插件市场安装

在Napcat等OneBot V11平台增加`HTTP服务器`，并配置好端口号，默认为`http://127.0.0.1:3000`

安装完成后通过命令`点歌`触发
